== Description ==

This plugin used for add edit delete and listing eBooks at admin side.Also user can search and sort records.
There is also short code implementation for client side.


A few notes about the sections above:

*   Gustav
*   add, edit, delete, listing, searching
*   "2.0" is the lowest version that the plugin will work on
*   "4.6" is the highest version that you've *successfully used to test the plugin*. Note that it might work on
higher versions... this is just the highest one you've verified.

== Installation ==

1. Upload myplug folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. For display listing page at front side use short code [vmember_List].
